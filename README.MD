2. 정렬(Sort) 
  - 정렬은 선택정렬 삽입정렬 버블정렬 퀵정렬이 있다. 
    1) 선택 정렬은 어떠한 기준이 되는 수를 잡고 배열의 첫번째부터 비교하여 , 내림차순으로 정렬한다.
    2) 삽입 정렬은 어떠한 기준이 되는 수를 잡고 위와 같이 오름차순으로 정렬한다. 
    3) 버블 정렬은 어떠한 기준이 되는 수를 잡고 옆에 있는 숫자와 비교해서 대소를 구분한뒤 자리를 바꿔준다,
    4) 퀵정렬은 어떤 기준값을 잡고 왼쪽으로는 그보다 작은값 오른쪽으로는 그보다 큰 값으로 나열한다.
